# Setup Guide

## Getting Started with CloudSync Pro

Welcome to CloudSync Pro. This guide walks you through initial setup, from account creation to your first project.

## Step 1: Create Your Workspace

After signing up at app.cloudsyncpro.com/signup, you will be prompted to create a workspace:

1. Enter your **Workspace Name** (e.g., your company or team name)
2. Choose a **Workspace URL** (e.g., `yourcompany.cloudsyncpro.com`). This cannot be changed later.
3. Select your **Industry** and **Team Size** to customize default templates
4. Click **Create Workspace**

## Step 2: Configure Workspace Settings

Navigate to **Settings > Workspace** to configure:

- **Timezone**: Set the default timezone for all deadlines and scheduled events
- **Work Week**: Define working days (default: Monday-Friday) for timeline calculations
- **Default Project Template**: Choose from Agile, Waterfall, Kanban, or Blank
- **Notification Preferences**: Set default notification channels (email, in-app, Slack)

## Step 3: Invite Team Members

1. Go to **Settings > Members > Invite**
2. Enter email addresses (comma-separated for bulk invites, up to 50 at once)
3. Assign roles: **Admin**, **Member**, or **Viewer**
4. Click **Send Invitations**

Invitees receive an email with a link to join. Invitations expire after 7 days. Resend from **Settings > Members > Pending Invitations**.

## Step 4: Create Your First Project

1. Click the **+ New Project** button in the sidebar
2. Choose a template or start from scratch
3. Enter the project name, description, and set a start/end date
4. Add team members to the project
5. Create your first task by clicking **+ Add Task** in any board column or list view

## Step 5: Connect Integrations (Optional)

Go to **Settings > Integrations** to connect tools your team already uses:

- **Slack**: Real-time notifications in channels
- **GitHub**: Link commits and PRs to tasks
- **Google Drive**: Attach files directly from Drive
- **Jira**: Two-way sync for teams migrating from Jira

## Minimum System Requirements

- Modern web browser (Chrome 90+, Firefox 88+, Safari 15+, Edge 90+)
- Stable internet connection (minimum 1 Mbps)
- Screen resolution: 1280x720 or higher
